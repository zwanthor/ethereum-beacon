Using Ethereum as a Random Number Beacon

This is a project to use the Ethereum Block chain as a source of public randomness.

The inspiration comes from https://eprint.iacr.org/2015/1015.pdf

Current Commands:
- python beacon.py <time>
  - produce random string ever set amount of time in seconds
- python interface.py <flag> [flag options]
  - produce random string using criteria from flag and its set options
